## 项目介绍-PureDataBase

## 项目进度

### 2026.04.07

初始化项目并完成REPL原型。

### 2026.04.08

将项目优化，抽出不同类型的操作命令，并实现世界上最简单的SQL编译器和虚拟机。

### 2026.04.09

实现数据库表硬编码。

通常数据库的底层实现是通过B树，在这里为了简单实现，直接使用数组存储页。一个页中存放多个数据库表行数据。

进行测试，发现在插入时的字符串会导致移除问题，所以需要对解析器进行优化，防止超出系统设计的阈值。

处理id可以为负这种情况，在判断字符串长度之前再添加一个判断id取值范围的逻辑则可。

### 元命令

不是SQL语句的命令，用于退出执行等操作。

例如：`.quit`、`.table`等。
